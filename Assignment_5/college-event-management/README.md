# College Event Management System

This is a React-based web application for managing college events, including viewing clubs, event cards, details, registration, and navigation.

## Features
- Event listing with cards
- Event details view
- Registration form
- Club information
- Responsive navbar

## Quick Start
1. Clone the repo: `git clone https://github.com/Shraddhawakchaure3/Web-Technology-Lab.git`
2. cd into `college-event-management`
3. Install dependencies: `npm install`
4. Start the dev server: `npm start`

## Deployment
Built with Create React App. Use `npm run build` for production builds.
